# Video to MP3 Converter

A simple Python application that extracts audio from video files and saves it as an MP3 file.

## Features

* Select any video file from your computer.
* Extract audio from videos.
* Save audio in MP3 format.
* Easy-to-use file selection dialog.
* Beginner-friendly Python project.

## Technologies Used

* Python
* MoviePy
* Tkinter

## Installation

```bash
pip install moviepy
```

## How to Run

```bash
python main.py
```

## Workflow

1. Select a video file.
2. Choose the location to save the MP3 file.
3. The program extracts the audio.
4. MP3 file is generated successfully.

## Learning Outcomes

* Working with multimedia files in Python.
* Using MoviePy for audio/video processing.
* File handling with Tkinter dialogs.
* Building practical automation tools.
